# Заметки - PWA с WebSocket и Push-уведомлениями

Офлайн-приложение для управления заметками с синхронизацией в реальном времени.

|----------|----------|

| **Институт** | ИПТИП |

| **Преподаватели** | Загородних Н.А., Краснослободцева Д.Б. |
